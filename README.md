# gitconfig Example
Simply clone this project, rename gitconfig file to ~/.gitconfig.

The __ diff algorithm __ will be setted to __ histogram __

The following __ aliases __ are configured :

    st = status
    sb = show-branch
    co = checkout
    ci = commit
    lg = log
    dword = diff --word-diff --unifie=10
    dfiles = diff --stat
    so = show object
